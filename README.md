# Ley-de-Coulomb-Bloques
Programa que calcula la fuerza neta eléctrica ejercida sobre una carga puntual objetivo debido a n cargas puntuales distribuidas en el plano cartesiano.

El sistema aplica la Ley de Coulomb para determinar la contribución de cada carga y luego suma vectorialmente todas las fuerzas para obtener:
El vector de fuerza neta: ⟨Fx, Fy⟩ (en Newtons)
La magnitud de la fuerza neta
